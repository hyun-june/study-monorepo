# 📦 Frontend Monorepo

개인 학습 코드와 이전 프로젝트를 체계적으로 관리하기 위한 프론트엔드 모노레포입니다.

여러 레포로 분산되어 있던 코드를 하나의 저장소로 통합하여
학습 과정과 코드 흐름을 함께 관리하는 것을 목표로 합니다.

---

## 📁 Structure

```
.
├── study/           # 강의 및 커리큘럼 기반 학습 코드
│   └── coding-noona/
│   └── lecture/
├── portfolio/     # 포트폴리오 사이트 코드
├── archive/         # 이전 프로젝트 및 참고용 코드
```

---

## 📌 About

이 레포는 다음 기준으로 코드를 분류합니다:

- **study**: 강의 및 학습 기반 코드
- **portfolio**: 포트폴리오 사이트 코드
- **archive**: 과거 프로젝트 및 참고용 코드

---

## 🛠 Tech Stack

- React
- TypeScript
- JavaScript
- HTML / CSS

---

## 🎯 Purpose

- 학습 과정과 결과물을 한 곳에서 관리
- 코드 구조를 정리하여 가독성 향상
- 점진적으로 프로젝트 단위로 확장
